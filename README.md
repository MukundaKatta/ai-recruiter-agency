# AI Recruiter Agency

> AI-Powered Recruitment Automation

## Overview

An intelligent recruitment platform that automates candidate screening, skills matching, interview scheduling, and hiring pipeline management. AI analyzes resumes, conducts initial screenings, and ranks candidates by fit.

## Key Features

- **Resume Parsing** — AI extracts skills, experience, and qualifications
- **Candidate Matching** — Score candidates against job requirements
- **Automated Screening** — AI-conducted initial phone/chat screenings
- **Interview Scheduling** — Calendar integration with automatic scheduling
- **Pipeline Management** — Kanban-style hiring pipeline tracking
- **Bias Detection** — Flag potential bias in job descriptions and screening
- **Analytics** — Time-to-hire, source effectiveness, pipeline conversion

## Tech Stack

Python, FastAPI, Claude API, PostgreSQL, React, Celery

## Getting Started

```bash
git clone https://github.com/MukundaKatta/ai-recruiter-agency.git
cd ai-recruiter-agency && pip install -e .
```
